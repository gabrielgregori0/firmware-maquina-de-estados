# Firmware com Máquina de Estados

Exemplo simples de firmware organizado por máquina de estados,
aplicável a sistemas embarcados como Arduino e ESP32.

## Estados implementados
- INIT: inicialização do sistema
- READ_SENSOR: leitura de dados
- PROCESS_DATA: processamento das informações
- SEND_DATA: envio via comunicação serial
- IDLE: estado de espera

## Objetivo
Demonstrar organização de firmware e controle de fluxo
utilizando máquina de estados em sistemas embarcados.
